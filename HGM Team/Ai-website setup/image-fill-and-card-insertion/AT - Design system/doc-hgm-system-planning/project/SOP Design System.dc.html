Design system

SOPs & step-by-step guides, built from one kit.

A shared component library for the documents we hand to clients and teammates. Every guide is assembled from three blocks — doc-header, doc-body, and doc-footer — with a set of modular, drag-and-drop variables inside the body. The team writes in the builder; clients read, check off steps, and sign off.
